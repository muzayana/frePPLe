==============
Batch commands
==============

Several executables are available to use frePPLe from the command line.

.. toctree::
  :maxdepth: 2
  :hidden:

  frepplectl
  frepple
  freppleservice


* | :doc:`frepplectl`
  | This is the main command line executable.
  | It runs administration commands for the database layer user interface.

* | :doc:`frepple`
  | This executable runs the planning engine from the command line.
  | Using this executable is only required for low-level operations, or when
    embedding frePPLe into another application.

* | :doc:`freppleservice`
  | This executable is available only on Windows, and controls the service
    that runs the web server for the user interface.
